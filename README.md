# TorchServe_Huggingface_example

Learning how to make a Huggingface handler for TorchServe

# Usage:

```bash
./0.download-model-from-huggingface.sh

./1.make-archiver.sh

./2.start-torchserve.sh
```
Then, wait until the server started.
Open another terminal:
```bash
./3.register-model.sh
```
